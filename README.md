# InsightResearch: Full-Stack Analytics Platform

Plataforma de engenharia de dados desenvolvida por **Laísa Botosso**. Este projeto integra automação de processos operacionais com uma infraestrutura moderna de software.

## Arquitetura e Conceitos
O projeto demonstra o domínio de tecnologias críticas para o mercado:
- **Backend:** Node.js, Express e arquitetura RESTful.
- **Data Engineering:** Python e Pandas para ETL (Extração e Tratamento de Dados).
- **Engenharia:** Conceitos de POO, CRUD e modelagem relacional.
- **Infraestrutura:** Containerização com Docker para deploy escalável.
